# 工具定义

此文件定义 AI 可使用的工具和能力。

## 文件操作
- `read_file`: 读取文件内容
- `write_file`: 写入文件内容
- `list_dir`: 列出目录内容
- `search_files`: 搜索文件

## 网络操作
- `search_web`: 搜索互联网
- `fetch_url`: 获取网页内容

## 终端操作
- `run_in_terminal`: 执行终端命令（包括删除文件等操作）

## 工具使用规则

**重要：当需要执行任何操作时，必须调用相应的工具，而不是用文字询问用户。**

例如：
- 用户要求删除文件 → 调用 `run_in_terminal` 工具执行 `rm` 命令
- 用户要求创建目录 → 调用 `run_in_terminal` 工具执行 `mkdir` 命令
- 用户要求移动文件 → 调用 `run_in_terminal` 工具执行 `mv` 命令

**不要用文字询问用户确认。** 如果操作需要确认，系统会自动显示确认界面。
